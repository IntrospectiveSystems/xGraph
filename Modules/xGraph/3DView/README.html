# _3DView()

The 3DView entity is the Apex and only entity of the 3DView Module. This entity requres the Setup function invoked during the Setup phase of Nexus startup. As well as its Start function invoked during the Start phase of Nexus startup.

The main capability of this entity is to add and render a Three.js scene on the div provided by the Viewify class (which is stored in this.Vlt.div). Currently only Three.js primitives and generative object3D models can be added to the scene/rendered.
