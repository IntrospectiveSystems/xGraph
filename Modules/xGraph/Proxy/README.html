# Setup(com, fun)

Determines which role Proxy is operating as (Client or Server). Role defined in Par.Role Starts Server or connects to defined external server as set in Par.

# Proxy(com, fun)

Routes commands depending on role defined in Par.Role Client:: Routes to module defined in Par.Link. Server:: Routes to TCP connection defined in Par.Host and Par.Port.

Parameters

Name Type Description
com Object

Command to be sent through Proxy
fun function

Callback function returned after command is received and processed at it's destination.

Returns

Void
