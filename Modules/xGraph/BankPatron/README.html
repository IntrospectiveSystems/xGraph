# BankPatron()

The BankPatron entity provides the user with a text interface to interact with a BankAccount. The user is presented with 3 options: check balance, deposit funds, withdraw funds.

Parameters

Name Type Description
this.Par.BankAccount pid

The pid to send the UserCommand too, a BankAccount.

Returns

Void

# Start(com)

Start sends a command to the BankAccount module (this.Par.BankAccount). When it receives a callback from BankAccount with the command argument Ready set to true, BankPatron dispatches it's StartUserInteraction command, which launches the text interface.

Parameters

Name Type Description
com object

The command object.

Returns

Void

# StartUserInteraction(com)

StartUserInteraction starts the text user interface for interacting with the BankAccount module. Users are presented with three options: Check Balance, Deposit Funds or Withdraw Funds. If the user chooses to Deposit or Withdraw funds, the user is then prompted to enter an amount.

Parameters

Name Type Description
com object

The command object.

Returns
